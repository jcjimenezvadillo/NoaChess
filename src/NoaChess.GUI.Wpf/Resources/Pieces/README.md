# Chess piece images

Classic "Cburnett" chess piece set by [Colin M.L. Burnett](https://en.wikipedia.org/wiki/User:Cburnett),
downloaded from [Wikimedia Commons](https://commons.wikimedia.org/wiki/Category:SVG_chess_pieces).

Licensed under [CC BY-SA 3.0](https://creativecommons.org/licenses/by-sa/3.0/) / GFDL.

The original SVG vectors are used directly (rendered via SharpVectors), so the
pieces rescale losslessly with the board.

File naming: `<piece><color>.svg` where piece is `k q r b n p`
(king, queen, rook, bishop, knight, pawn) and color is `l` (light/white) or `d` (dark/black).
